# Changelog

All notable changes to this project will be documented in this file.

## [1.0.2] - 2026-06-18

### Added
- Added a visible plugin version label in the admin dashboard header.
- Added `WIKI.md` quick-reference documentation.

### Changed
- Bumped plugin version metadata to `1.0.2`.
- Updated `README.md` and `readme.txt` to align with current checks/fixes and release notes.

## [1.0.1] - 2026-06-18

### Changed
- Removed the `PHP execution blocked in /uploads/` hardening item from checks and one-click fixes.
- Updated plugin version metadata to `1.0.1`.
- Updated `readme.txt` stable tag and release notes for `1.0.1`.
- Removed the outdated File Security feature bullet that referenced uploads PHP execution blocking.

## [1.0.0] - 2026-06-18

### Added
- Initial release of Hostnasi Security Hardening.
- Security hardening dashboard with category-based checks and pass/fail scoring.
- One-click fix actions for supported hardening items.
- Runtime protections including login lockout, XML-RPC disable, security headers, user enumeration blocking, and hidden login URL support.
